Propose a splinter session

Submission of splinter session proposals will be handled online using a web form provided by the Google Forms platform. The proposal form can be accessed via the following link:

http://goo.gl/forms/muZeZ5B5H3

We advise preparing responses and compiling information prior to accessing the web form. Below is a list of all the requested information and a description of each item. A website template consistent with the main Cool Stars 19 webpage is available on GitHub by forking the main repository. A printer-friendly PDF version of the instruction can be found via this link.

Deadline is 16 Oct 2015.

Preliminary information

List of names and affiliations of co-organizers.

Webpage

URL of the splinter session webpage, if available.

Note that a valid web URL is not mandatory at this point in time. However, if you do not provide a URL, you must state the anticipated web address in the “Webpage Commitment” section below.

Scientific justification

Fields marked with an asterisk “*” are mandatory

Field

Description

Scientific Motivation ​*

Discuss the aim(s) of the splinter session, why it is timely (in particular, why should it be convened at Cool Stars 19?), and the broader impact of the session. Length should be approximately two or three paragraphs, but no more than one page.

Potential Invited Speakers *

If you plan to have invited speakers for the splinter session, please provide a tentative list of these speakers and their institutional affiliation. Note that this list is ​non-binding ​

Potential Attendees ​*

Whom do you foresee being interested in attending your proposed splinter session? Please provide a list of names and institutions.
